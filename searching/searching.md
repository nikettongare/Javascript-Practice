## Searching Algorithums

- linear / sequential search
  - when array is unsorted.
  - check every element one by one untile search found.
  - big O
    - best case : O(1)
    - average : O(n)
    - worst : O(n)

<br>

- binary search
  - when array is sorted.
  - divide and conquer method.
  - faster and linear search.
  - big O
    - best : O(1)
    - worst : Olog(n)
